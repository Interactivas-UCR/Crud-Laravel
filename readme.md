## Iniciar

Cambiar el nombre del archivo .env.example a .env
Agregar los campos:

- DB_DATABASE=
- DB_USERNAME=
- DB_PASSWORD=

Ejecutar en la terminal: 

```bash 
    composer require && npm i 
```

Se debe de ejecutar:

```bash 
    php artisan key:generate
```
